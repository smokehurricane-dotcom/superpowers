## Aufnahme-Setup + Tastenkürzel

**Tastenkürzel (Xbox Game Bar):**
- `Win + G` — Game Bar öffnen
- `Win + Alt + R` — Aufnahme **Start/Stop** per Taste (KEINE Maus zum Button → nichts wegzuschneiden)

**Fenster-Aufnahme (Skript bleibt unsichtbar):**
- Game Bar filmt **nur das aktive Fenster** = das Dashboard. Ein Skript-Fenster *daneben* wird **nicht** mitgefilmt. ✅
- Skript in einem **eigenen Fenster/App** (z.B. Notepad) öffnen — **NICHT** als zweiter Tab im selben Browser.
- Während der Aufnahme **nicht ins Skript klicken** — Dashboard fokussiert lassen (du tippst nur dort), Skript mit den Augen lesen.
- Dashboard-Hälfte **breit genug** machen, damit's im Video gut lesbar ist.
- **Erst 5-Sek-Testaufnahme** → abspielen → prüfen, dass NUR das Dashboard im Bild ist. Dann die echten Takes.

## Ablauf
1. `streamlit run app_shk.py` starten, Dashboard-Fenster sauber.
2. Demo-Frage einmal zur Probe stellen.
3. Szene für Szene aufnehmen (`Win+Alt+R`), zwischen den Clips kurz sammeln.
4. In Clipchamp aneinander, Kopf/Ende-Totframes trimmen → MP4.
5. MP4 als **YouTube „nicht gelistet"** hochladen → Link in die Mail.
